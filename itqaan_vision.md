## ١. نظرة عامة على المشروع

منصة إتقان هي منصة تعليمية إسلامية متكاملة، انطلقت كأداة لتقييم تلاوة سورة الفاتحة، والهدف الآن تحويلها إلى بيئة تعليمية شاملة تغطي حفظ القرآن الكريم كاملاً، الدروس الشرعية، جلسات التعليم الفردي والجماعي، وبناء مجتمع إسلامي متعلم ومتفاعل.

> **الرؤية: أن تكون إتقان المرجع الأول رقمياً لكل من يريد تعلم القرآن والعلوم الشرعية بشكل منظم ومعتمد.**

المنصة تعمل حالياً على ثلاثة محاور: الطالب يسجل تلاوته → القارئ يراجعها → يُحجز موعد عند الحاجة. النسخة الجديدة ستوسع كل محور بشكل جذري وتضيف محاور جديدة كاملة.

## ٣. المميزات الجديدة الكاملة

> 📌 كل ميزة ستُبنى فوق ما هو موجود حالياً بدون هدم ما يعمل — التطوير تراكمي.

### ٣.١  توسيع محتوى القرآن الكريم

المنصة ستدعم القرآن كاملاً — ١١٤ سورة — بدلاً من الاقتصار على الفاتحة. الطالب يقدر يسجل أي آيات يريد مع تحديد السورة والأرقام بدقة.

### ٣.٢  ربط API القرآن والتسجيلات الصوتية

ربط المنصة بـ API متخصص للقرآن الكريم يوفر نص كل آية بالرسم العثماني، والتسجيلات الصوتية لكل آية بأصوات قراء مختلفين كمرجع للطالب والقارئ.


| API / الخدمة | الاستخدام في المنصة |
|---|---|
| Quran.com API | جلب نص الآيات بالرسم العثماني وعلامات التجويد، تحديد السور والأجزاء والأحزاب تلقائياً |
| MP3Quran / EveryAyah | تشغيل التسجيل الصوتي لكل آية بصوت القارئ المختار كمرجع للمقارنة |
| Aladhan API (أوقات الصلاة) | جلب أوقات الصلاة حسب موقع الطالب وربط التذكيرات بها |
| Quran Word API | عرض كل كلمة قرآنية مع معناها وإعرابها — مفيد في دروس التفسير |

### ٣.٣  نظام المسارات التعليمية

بدلاً من أن الطالب يسجّل بشكل عشوائي، ستوجد مسارات تعليمية منظمة تأخذه خطوة خطوة من البداية للختم أو من مستوى لمستوى.

### ٣.٤  تتبع الحفظ اليومي

نظام يتيح للطالب تسجيل تقدمه اليومي في الحفظ والمراجعة، مع مخطط بياني يوضح التقدم على مر الأسابيع.

### ٣.٥  الدروس والعلوم الشرعية

إضافة قسم كامل للتعليم الديني بجانب تلاوة القرآن — الشيخ يرفع دروساً ويفتح جلسات مباشرة.

### ٣.٦  نظام المهام والتقويم

نظام متكامل يجمع مهام الطالب اليومية مع تقويم بصري يعرض كل مواعيده وخططه في مكان واحد.

### ٣.٧  الفيديو كول المدمج

بدلاً من الاعتماد على روابط Zoom خارجية، سيكون فيديو كول مدمج داخل المنصة تماماً — الطالب يدخل جلسته بضغطة واحدة.

### ٣.٨  نظام النقاط والمستويات

نظام تحفيز شامل يجعل الطالب يلتزم ويستمر — كل فعل في المنصة يكسب نقاطاً ويفتح مستويات وشارات.

### ٣.٩  نظام الشارات والإنجازات

شارات بصرية تظهر في ملف الطالب وتُشارَك مع المجتمع — كل إنجاز له شارة خاصة.

### ٣.١٠  المسابقات الدورية

مسابقات تحفيظ وتجويد تُقام دورياً — تعزز روح المنافسة الإيجابية وتحفز الطلاب على الاستمرار.

### ٣.١١  لوحة ولي الأمر

حساب منفصل لأهل الطالب الصغير يتيح لهم متابعة تقدمه ومستواه بشكل مستمر.

### ٣.١٣  لوحة الإحصائيات الوطنية

لوحة خاصة بإدارة المنصة تعرض انتشار المنصة جغرافياً ومعدلات النشاط حول العالم.

### ٣.١٤  تحسين التصميم والتجربة

المنصة الحالية تحتاج تحديث بصري شامل يعكس الهوية الإسلامية الحديثة ويرفع تجربة المستخدم.

### ٣.١٥  فصل التعليم حسب الرواية القرآنية

كل رواية قرآنية لها قواعد تجويد وأحكام مختلفة — المنصة تفصل بينهم لضمان صحة التعليم.

### ٣.١٦  الدروس العامة المفتوحة للجمهور

الشيخ يقدر يفتح دروساً عامة برابط قابل للمشاركة على السوشيال ميديا — يدخل عليها أي شخص بدون حساب على المنصة، وبعدها يمكنه الاشتراك.

### ٣.١٧  فحص القراء والمدرسين قبل القبول

لضمان الجودة، كل قارئ أو مدرس جديد يمر بعملية فحص وتحقق قبل أن يُقبَل على المنصة.

### ٣.١٨  قسم الأسئلة الفقهية

قسم مخصص للأسئلة الشرعية — الزوار يسألون والمسؤولون يردون، مع نشر الإجابات كمرجع للجميع.

### ٣.١٩  المنتدى والمجتمع والمقالات

قسم مجتمعي يجمع الطلاب والقراء في مساحة للتعلم المشترك والنقاش والمقالات.

### ٣.٢٠  نظام الدعوات

الأدمن يقدر يدعو مستخدمين مباشرة عبر الإيميل ويضعهم في خطة معينة من اليوم الأول — بدون ما الشخص يمر بعملية التسجيل العادية.
